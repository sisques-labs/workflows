# GitHub Pages Documentation

This directory contains the static website for GitHub Pages.

## Setup

1. Go to your repository settings on GitHub
2. Navigate to **Pages** in the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Choose the branch (usually `main`) and set the folder to `/docs`
5. Click **Save**

The site will be available at `https://<username>.github.io/workflows/` or `https://<org-name>.github.io/workflows/`

## Files

- `index.html` - Main HTML page
- `styles.css` - CSS stylesheet
- `script.js` - JavaScript for interactivity
- `_config.yml` - Jekyll configuration (optional, only needed if using Jekyll)

## Customization

You can customize the site by editing:
- Colors and styling in `styles.css`
- Content and structure in `index.html`
- Interactive features in `script.js`
